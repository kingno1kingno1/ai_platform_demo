
# AI 平台 Demo 說明文件（示範）

這份文件主要是為了配合 RAG demo 使用，你可以在這裡填入：

- 你為什麼做這個 side project（例如：配合某個面試 JD 的需求）。
- 系統的整體架構（可以搭配架構圖）。
- 你在這個專案裡實作了哪些功能：
  - Multi-agent 協作
  - RAG 檢索
  - 本地 LLM 推理與 GPU 使用
  - Prometheus 監控
- 以及你未來如果要擴展到：
  - 多 GPU / 多節點
  - KServe / Triton / Ray Serve
  - 更多工具與 agent
  - 更嚴謹的 SLO / SLA

在面試時，你可以把現在這個專案當作「縮小版的平台」，說明你已經瞭解概念，
只是資源規模與生產環境會更大，但方向是相同的。
